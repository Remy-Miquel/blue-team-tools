# Checklist publication

## Sécurité des données

- [ ] Le projet est dans un environnement légal et autorisé
- [ ] Les données sensibles sont supprimées
- [ ] Les captures sont anonymisées
- [ ] Les IP privées/école/client sont masquées si nécessaire
- [ ] Aucun mot de passe n'est visible
- [ ] Aucun token ou cookie n'est visible
- [ ] Aucun nom d'étudiant ou professeur n'est visible
- [ ] Les flags sont masqués si la plateforme l'exige

## Qualité du dépôt

- [ ] Le README explique le contexte
- [ ] Le README explique les compétences démontrées
- [ ] Le rapport contient des remédiations
- [ ] Le projet est compréhensible par un recruteur non expert
- [ ] Les fichiers inutiles sont supprimés
- [ ] Le dépôt a une description GitHub claire
- [ ] Les topics GitHub sont renseignés
